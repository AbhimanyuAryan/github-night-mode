Install

To install the extension, download the sourcecode, visit chrome://extensions in Chrome and drag the folder containing the above files into the window.

Alternatively, you can follow the below process as well.

  1. Install the Stylish extension for Chrome.
  2. Open up extension options and paste the whole CSS.
  3. Specify the domain name to be github.com.
  4. Add a title and save.

Boom! You have your all new GitHub in night mode. ðŸŽ‰
